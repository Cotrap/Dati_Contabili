Scaricare i Dati Contabili

Segui questi passaggi per generare e scaricare il file dei dati contabili relativi alle Spese Abbonamenti Agenzie.

🔐 Profilo Contabile 🧾 Ordini → Spese Agenzie 📤 Esportazioni
⚡ In breve
  • Accedi con le credenziali del Profilo Contabile (diverse da quelle commerciali).
  • Vai in Ordini → Spese Abbonamenti Agenzie, inserisci l'anno e premi Esporta.
  • Recupera il file da Esportazioni: l'ultima richiesta è sempre in cima alla lista.
📋
Panoramica

Introduzione

Questa procedura permette di estrarre dal portale Itineris i dati contabili relativi alle Spese Abbonamenti Agenzie per un determinato anno. Il file generato può essere importato nei software di gestione aziendale.

⚠️
Attenzione: assicurati di essere nel Profilo Contabile e non in quello commerciale. Controlla il nome utente visualizzato in basso a sinistra della schermata: deve corrispondere alle credenziali contabili fornite da COTRAP. Se sei nell'account sbagliato, esegui il logout e accedi con le credenziali corrette.
🚀
Guida Passo-Passo

Procedura Completa

1
🔐 Accedi al Profilo Contabile

Accedi al portale Itineris usando le credenziali contabili (ID utente e password forniti dall'Amministratore COTRAP). Le credenziali contabili sono diverse da quelle del profilo commerciale.

Verifica sempre di essere nell'account corretto: in basso a sinistra della schermata è visibile il nome utente e tra parentesi il nome dell'azienda consorziata.

2
🧾 Entra in Ordini → Spese Abbonamenti Agenzie

Dal menu principale in alto, clicca su Ordini.
Si aprirà il sottomenu: seleziona la voce Spese Abbonamenti Agenzie.
Si aprirà la sezione dedicata con i filtri di ricerca.

3
📅 Inserisci l'Anno Contabile

Nel campo Anno Contabile inserisci l'anno di cui hai bisogno (es. 2025).
Verifica che tutti gli altri filtri siano impostati correttamente prima di procedere.

4
⬆️ Avvia l'Esportazione

Clicca sul pulsante Esporta.
Il sistema elaborerà la richiesta e preparerà il file. Non chiudere la pagina durante l'elaborazione.
Il file non viene scaricato subito: verrà reso disponibile nella sezione Esportazioni.

5
📤 Vai alla Sezione Esportazioni

Dal menu principale, clicca su Esportazioni.
Si aprirà l'elenco di tutti i file generati dal profilo contabile.
L'ultima esportazione richiesta apparirà in cima alla lista (la più recente è sempre la prima).

6
⬇️ Scarica il File

Individua l'esportazione in cima alla lista e clicca sull'icona di download a destra della riga.
Il file verrà scaricato automaticamente nella cartella Download del tuo PC.

ℹ️
I file non vengono cancellati automaticamente dalla sezione Esportazioni. È possibile tornare in qualsiasi momento per scaricare nuovamente un file già generato in precedenza.
✅
Operazione completata! Il file dei dati contabili è ora disponibile nella cartella Download del tuo PC, pronto per essere importato nel gestionale aziendale.
